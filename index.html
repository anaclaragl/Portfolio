Game Description

Lune is a game about a white moth on a quest to retrieve light, restoring the moon's glow for all mothkind. By carrying a lantern that changes color, the moth can interact with the world around it.

Dialogue Tool

2025

A useful tool to create dialogues.

Tool Description

While working on Rangolitico, I designed and developed a Editor Tool to create dialogue trees without relying on code. This tool provides a visual node-based windown within Node Graph and Editor Window.

Features
  • Visual Node-Based Editor - Drag-and-drop creation of dialogue branches, choices, and conditions.
  • Data-Driven Design - Exports clean, serializable data structures compatible with Unity's ScriptableObjects for runtime use.
